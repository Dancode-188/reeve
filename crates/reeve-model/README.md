# reeve-model

The vocabulary. Entities (agents, traces, spans, commands, policy
rules), typed IDs, and the signal enums every other Reeve crate
speaks. No IO, no logic beyond the types; everything depends on this
crate and this crate depends on nothing of Reeve's.

Part of [Reeve](https://github.com/Dancode-188/reeve), the terminal
cockpit for AI agents.
