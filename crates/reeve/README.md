# reeve-cockpit

The Reeve binary. Installs as `reeve`: a terminal cockpit that watches
AI agents work, scores what they do, and intervenes when they go
sideways. This crate wires the pipeline, the engine, the control
server, and the renderer into one process and owns startup, the three
loopback ports, and retention scheduling.

Start here: [the repository](https://github.com/Dancode-188/reeve),
which has the full README, guides, and architecture.
