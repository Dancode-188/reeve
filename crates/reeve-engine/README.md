# reeve-engine

The judgment. Heuristic evaluators, the optional Ollama-backed LLM
judge, the 0 to 100 health score, the policy rule engine, per-agent
daily budgets with store-backed resync, and outcome tracking that
measures whether an intervention actually helped.

Part of [Reeve](https://github.com/Dancode-188/reeve), the terminal
cockpit for AI agents.
